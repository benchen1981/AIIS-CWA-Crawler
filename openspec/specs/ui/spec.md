## Requirements
### Requirement: CWA Visual Theme
The app MUST implement a visual theme consistent with the CWA website.

#### Scenario: Visual Match
-   Given the app is open
-   When the user views the main page
-   Then the header should use CWA blue and white colors.
-   And the table should have clear, styled headers.

### Requirement: Table Columns
The app MUST display columns consistent with the CWA observation table.

#### Scenario: Column Names
-   Given the weather table
-   When displayed
-   Then columns should be labelled: "縣市", "地點", "溫度", "觀測時間" (or similar CWA labels).
